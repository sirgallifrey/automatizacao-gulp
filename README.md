## Running exemples:

First install gulp and bower globally

```
npm install -g gulp-cli bower
```

then navigate to desired folder and install dependencies
```
cd EXEMPLE
npm install & if [ -e bower.json ] ; then bower install ; fi
```

Run it
```
gulp
```
